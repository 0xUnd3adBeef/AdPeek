# Code of Conduct

This project is for **authorized security testing and research** involving Active Directory.  
There is only one rule: **use this tool legally and responsibly**.

---

## Authorized Use Only

You may use or contribute to this project **only** if:

- You have explicit permission to test the environment you target  
- You understand and follow all applicable laws  
- You accept full responsibility for your actions  

If you don’t have permission to touch a system, **don’t use this tool on it**.  
Nothing in this repo is intended for, or should be used for, unauthorized compromise.

---

## Contributions

When contributing:

- Do not submit code designed for unauthorized exploitation  
- Do not include real credentials, internal domain data, etc.  
- Only contribute techniques relevant to **legitimate AD security assessment**  
- Keep submissions clean, functional, and technically accurate  

---

## No Liability

The maintainers are **not responsible** for how you use this software.  
If you break something, get in trouble, or misuse the tool, **that’s on you**.

---

## Enforcement

If you submit or discuss anything that clearly promotes illegal use,  
your issue/PR will be closed immediately.
